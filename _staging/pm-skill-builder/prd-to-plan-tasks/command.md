---
description: Convert an existing PRD into a practical delivery plan and implementation-ready task breakdown.
argument-hint: "[PRD text or file path]"
allowed-tools: Read, Glob, Grep
---

Use the skill at `skills/deliver-prd-to-plan-tasks/SKILL.md`.

Convert the provided PRD into:

- A source PRD summary
- Requirement coverage map
- Delivery plan with milestones and dependencies
- Implementation-ready task breakdown
- Validation and release readiness work
- Risks, mitigations, open questions, and definition of done

Keep all plan items traceable to the PRD where possible. If the PRD lacks
critical planning detail, make conservative assumptions and list open questions
separately.
