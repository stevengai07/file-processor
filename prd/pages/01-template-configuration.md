# 字段模板配置

> 路由建议：`/templates/new`  
> 模块：字段模板  
> 日期：2026-06-09

## 页面目标

用户上传一份 Excel 表格，将其中的行解析成待提取字段，并在开始批量处理前检查、编辑和确认字段规则。

## 页面布局

1. 顶部：步骤状态“字段模板 → 上传文档 → 提取结果”。
2. 模板上传区：拖放或选择 Excel 文件。
3. 模板解析区：显示工作表、表头识别结果和错误。
4. 字段编辑表：逐行编辑字段规则。
5. 底部操作区：取消、重新上传、确认字段并继续。

## 输入字段

| 字段 | 类型 | 必填 | 默认值 | 校验与说明 |
|---|---|---:|---|---|
| 模板文件 | 文件 `.xlsx` | 是 | 无 | 只能上传一个文件 |
| 工作表 | 下拉选择 | 是 | 第一个工作表 | 多工作表时允许切换 |
| 表头所在行 | 正整数 | 是 | 1 | 用于处理顶部含标题或说明的表格 |

## 字段定义表

| 列 | 类型 | 必填 | 规则 |
|---|---|---:|---|
| 字段名称 | 单行文本 | 是 | 模板内唯一，去除首尾空格后不能为空 |
| 字段说明 | 多行文本 | 否 | 用于告诉 AI 字段业务含义和提取依据 |
| 数据类型 | 下拉选择 | 是 | 文本、长文本、整数、小数、日期、布尔值、列表 |
| 是否必填 | 开关 | 是 | 默认否 |
| 示例值 | 单行文本 | 否 | 帮助模型理解格式，不作为固定答案 |
| 提取提示 | 多行文本 | 否 | 可包含同义词、位置、格式等提示 |
| 顺序 | 拖拽/数字 | 是 | 决定结果列和导出顺序 |

## 交互

### 上传模板

- **触发：** 用户选择 `.xlsx` 文件。
- **响应：** 系统读取工作表和表头，展示字段预览。
- **失败：** 文件损坏、无工作表或无有效字段时，显示可操作的错误信息。

### 自动映射列

- 系统优先识别“字段名称、字段说明、数据类型、是否必填、示例值、提取提示”等列名及常见同义词。
- 无法识别时，要求用户手动指定哪一列是“字段名称”。
- 当前需求未提供真实模板，具体同义词列表为 `[TBC]`。

### 编辑字段

- 用户可新增、删除、排序和修改字段。
- 删除字段需要二次确认，仅在字段已产生结果时要求确认重新提取。
- 字段名称重复、字段为空或类型无效时，确认按钮禁用。

### 确认并继续

- **触发：** 用户点击“确认字段并继续”。
- **响应：** 系统保存字段模板快照，进入批量上传页面。
- **规则：** 后续任务使用快照，避免源 Excel 被修改后影响任务。

## 建议模板格式

| 字段名称 | 字段说明 | 数据类型 | 是否必填 | 示例值 | 提取提示 |
|---|---|---|---|---|---|
| 合同编号 | 合同的唯一编号 | 文本 | 是 | HT-2026-001 | 可能写作合同号、协议编号 |
| 甲方名称 | 合同甲方完整名称 | 文本 | 是 | 某某有限公司 | 不要提取联系人姓名 |
| 合同金额 | 合同含税总金额 | 小数 | 否 | 100000.00 | 输出纯数字，币种另行提取 |

## API 依赖

| API | 方法 | 路径 | 用途 |
|---|---|---|---|
| 解析模板 | POST | `/api/templates/parse` | 上传并解析 Excel |
| 保存模板 | POST | `/api/templates` | 保存确认后的字段定义 |
| 获取模板 | GET | `/api/templates/{template_id}` | 恢复页面或创建任务 |

## 验收标准

- 可解析至少包含“字段名称”列的 Excel。
- 所有解析字段可在页面编辑和排序。
- 非法字段有明确行级错误。
- 确认后返回稳定的 `template_id` 和字段快照。
