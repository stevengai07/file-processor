# 页面关系

```text
字段模板配置
    |
    | 确认模板
    v
批量上传与处理
    |
    | 任务产生结果
    v
结果校对与导出
    |
    | 重试失败文件
    +--------------------> 批量任务状态更新
```

## 导航规则

| 来源 | 目标 | 条件 | 传递数据 |
|---|---|---|---|
| 字段模板配置 | 批量上传与处理 | 模板校验通过 | `template_id` |
| 批量上传与处理 | 字段模板配置 | 任务尚未启动 | 模板草稿 |
| 批量上传与处理 | 结果校对与导出 | 至少一份文件有终态 | `task_id` |
| 结果校对与导出 | 批量上传与处理 | 查看进度或重试 | `task_id`、`file_id` |

## 数据耦合

- 字段模板决定结果页动态列、类型校验和导出顺序。
- 批量任务保存模板快照，模板后续修改不得影响既有任务。
- 结果编辑会更新文件状态和导出数据。
- 文件重试会替换该文件的 AI 原始结果，但不得静默覆盖已人工修改值；具体冲突策略为 `[TBC]`，建议重试前确认。
