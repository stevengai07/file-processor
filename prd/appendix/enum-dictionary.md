# 枚举字典

## 模板字段类型

| 值 | 中文 | 校验规则 |
|---|---|---|
| `text` | 文本 | 单行字符串 |
| `long_text` | 长文本 | 多行字符串 |
| `integer` | 整数 | 不含小数部分 |
| `decimal` | 小数 | 可转换为十进制数 |
| `date` | 日期 | 推荐规范化为 `YYYY-MM-DD` |
| `boolean` | 布尔值 | 是/否、true/false 等映射为统一值 |
| `list` | 列表 | 多值数组；Excel 中使用换行或分隔符展示 |

## 任务状态

| 值 | 中文 | 说明 |
|---|---|---|
| `draft` | 草稿 | 尚未启动 |
| `queued` | 等待中 | 已提交，等待工作进程 |
| `running` | 处理中 | 至少一个文件正在处理 |
| `completed` | 已完成 | 所有文件进入终态 |
| `cancelled` | 已取消 | 用户取消任务 |
| `failed` | 任务失败 | 任务级故障，区别于单文件失败 |

## 文件状态

| 值 | 中文 | 说明 |
|---|---|---|
| `pending` | 等待 | 尚未开始 |
| `processing` | 处理中 | 正在提取 |
| `success` | 成功 | 提取并校验通过 |
| `needs_review` | 待检查 | 必填缺失或类型错误 |
| `failed` | 失败 | 无法产生结果 |
| `cancelled` | 已取消 | 尚未执行即被取消 |

## OCR 预设

| 值 | 用途 |
|---|---|
| `scanner` | 平板扫描件 |
| `phone_camera` | 手机拍摄、倾斜或反光 |
| `faded` | 旧文档、低对比度 |
| `fax` | 传真或噪点较多的文档 |

## 支持的文档格式

| 扩展名 | MVP | 处理方式 |
|---|---:|---|
| `.pdf` | 是 | 原生文本优先，文本过少时 OCR |
| `.docx` | 是 | 读取段落文本 |
| `.xlsx` | 仅模板 | 解析字段定义，不作为待提取文档 |
| `.doc` | 否 | `[TBC]` |
| 图片 | 否 | `[TBC]` |

## 问题类型

| 值 | 中文 |
|---|---|
| `required_missing` | 必填字段缺失 |
| `type_invalid` | 数据类型不符合模板 |
| `value_ambiguous` | 发现多个可能值 |
| `source_not_found` | 未在原文中找到依据 |
| `extraction_error` | 模型输出解析失败 |
| `ocr_error` | OCR 失败 |
